## Decisions Made

- Plant species removed from parity.py inline dict — all plant species resolved via YAML registry; eliminates drift between registry and parity.py. Reference organisms (human, mouse, etc.) and parasites kept as slim inline maps since they serve different API contract requirements
- Default species uses exact registry key format 'Arabidopsis thaliana' not 'arabidopsis_thaliana' — the underscore form split into one token by Python's `.split()`, producing `["arabidopsis_thaliana"]` which does not match the registry key `"arabidopsis thaliana"` built from space-separated lowercasing
